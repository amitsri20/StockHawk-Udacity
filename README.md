# StockHawk-Udacity
Android App to display stock prices over time.

## Features
- User can add new quote and track its behaviour over time.
- Detail chart view to understand historical trend of quote.

## Libraries used
- Retrofit
- OkHTTP
- MPAndroidChart
- Glide
